# SDET Playwright Automation Project

## 📌 Overview
This project automates the **Kaleidoscope Applicant Application** process using **Playwright & TypeScript**.

## 🔧 Setup & Installation
1. Install dependencies:
   ```sh
   npm install
   ```
2. Run the tests:
   ```sh
   npm run register_test

   npm run Application_test
   ```
